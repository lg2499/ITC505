Addendum

This project, “Pirates of the Caribbean: Choose Your Own Adventure,” was created using HTML, CSS, and JavaScript. It includes 8 different endings and dynamic image updates based on user choices. I planned the story flow first, then used JavaScript objects to manage scenes and consequences. Debugging event listeners and DOM updates helped me better understand how to create interactive web experiences. I also used CSS for a simple and immersive pirate theme.
