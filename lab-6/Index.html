Addendum / Reflection

Problem I solved: I built an interactive Number Sorter tool. The user types a list of numbers, and the page (using JavaScript) sorts them ascending and descending, and also shows min/max.

How I solved it: I take the input string, split it at commas, convert each piece to a Number, filter out bad values, then use array.sort((a,b)=>a-b) for ascending and array.sort((a,b)=>b-a) for descending. I update the page using textContent.

Difficulties: Input validation. If the user types "5, hi, 2" that breaks sorting. I fixed it by checking isNaN() and showing an error message with a red warning instead of crashing.
