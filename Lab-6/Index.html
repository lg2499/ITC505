📝 Addendum

Problem I solved: I built an interactive Number Sorter. The user types comma-separated numbers and clicks a button. The page shows those numbers sorted from smallest to largest.

How I solved it: This page uses a Holy Grail layout (header, nav, main, ad, footer) made with CSS Grid. The JavaScript logic lives in script.js and runs on button click. The script:

  1. Reads the text box value.
  2. Splits the string by commas into an array.
  3. Trims and converts each piece to a number.
  4. Sorts numerically using (a,b) => a - b.
  5. Displays the sorted numbers on the page.

Difficulties and fixes: Default JavaScript Array.sort() sorts as strings, so "100" would come before "2". I fixed that with my own compare function. I also added validation: if the user types something that's not a number, I show a red error instead of crashing.
