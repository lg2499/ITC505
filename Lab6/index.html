📝 Addendum

Problem I solved: I built an interactive number sorter. The user types comma-separated numbers and clicks a button. The page shows those numbers sorted from smallest to largest.

How I solved it: The page uses a Holy Grail layout (header / left menu / main / ad / footer) with CSS Grid. The JavaScript is in script.js. On click, it:

  1. Reads the text box value.
  2. Splits by commas into a list.
  3. Trims spaces and converts strings to numbers.
  4. Sorts the numbers ascending using (a,b) => a - b.
  5. Prints the sorted list into the page.

Difficulties / fixes: Default JavaScript sort is string sort ("100" comes before "2"). I fixed that by passing a compare function. I also added error handling to show a red warning if the input has something that's not a number.
