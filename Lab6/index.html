📝 Addendum

Problem I solved: I built an interactive “Number Sorter” web tool. The user types any comma-separated numbers and clicks a button. The page shows those numbers sorted in ascending order.

How I solved it: This page uses a Holy Grail layout (header, left nav, main content, right ad, footer) built with CSS Grid. The JavaScript lives in script.js and is loaded into the HTML using a <script> tag. The script:

  1. Reads the text box on button click.
  2. Splits the string by commas into an array.
  3. Trims and converts each piece to a number.
  4. Sorts numerically (smallest → largest).
  5. Prints the sorted array to the page.

Difficulties and fixes: By default, JavaScript sorts as text, which is wrong for numbers (for example "100" would come before "2"). I fixed that using a custom compare function (a,b) => a - b. I also added error handling: if any value is not a valid number, the tool shows a red error message instead of breaking.
