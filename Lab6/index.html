Addendum

Problem I solved: I built an interactive Number Sorter & Stats Tool. The user types a comma-separated list of numbers, and the page calculates a bubble-sorted list, minimum, maximum, and average.

How I solved it: In app.js I: (1) Read the input field, (2) Converted the string into an array of numbers, (3) Used a Bubble Sort function I wrote (nested loops that keep swapping if out of order), (4) Computed min, max, and average, (5) Updated the DOM to display the results. The button click is the event that triggers all this.

Difficulties & fixes: - Handling bad input like "hi, 3, !!" caused NaN. I fixed it by validating every value and showing an error box instead of crashing. - Making layout match Holy Grail: I used CSS Grid with named areas (header, nav, main, ad, footer) and responsive rules. - The assignment footer needs a "Last updated" script, so I included the provided code below.
